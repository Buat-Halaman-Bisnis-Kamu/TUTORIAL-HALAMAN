Selamat datang di Halaman GitHub. Generator halaman otomatis ini adalah cara termudah untuk membuat halaman yang indah untuk semua proyek Anda. Buat konten halaman Anda di sini [menggunakan GitHub Flavoured Markdown] (https://guides.github.com/features/mastering-markdown/), pilih templat yang dibuat oleh desainer, dan publikasikan. Setelah halaman Anda dibuat, Anda dapat memeriksa cabang gh-halaman baru secara lokal. Jika Anda menggunakan GitHub Desktop, cukup sinkronkan repositori Anda dan Anda akan melihat cabang baru.

Template Desainer Kami telah membuat beberapa template yang tampan untuk Anda gunakan. Silakan dan klik 'Lanjutkan ke tata letak' untuk menjelajahinya. Anda dapat dengan mudah kembali mengedit halaman Anda sebelum menerbitkan. Setelah menerbitkan halaman Anda, Anda dapat mengunjungi kembali generator halaman dan beralih ke tema lain. Konten Halaman Anda akan dipertahankan.

Membuat halaman secara manual Jika Anda memilih untuk tidak menggunakan generator otomatis, dorong cabang bernama gh-pages ke repositori Anda untuk membuat halaman secara manual. Selain mendukung konten HTML biasa, GitHub Pages mendukung Jekyll, generator situs statis sederhana yang sadar blog. Jekyll membuatnya mudah untuk membuat header dan footer di seluruh situs tanpa harus menyalinnya di setiap halaman. Ini juga menawarkan dukungan blog cerdas dan fitur templating canggih lainnya.

Penulis dan Kontributor Anda dapat @ menyebutkan nama pengguna GitHub untuk menghasilkan tautan ke profil mereka. Elemen <a> yang dihasilkan akan ditautkan ke Profil GitHub kontributor.
